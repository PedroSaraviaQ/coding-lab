CSS selectors

Element selectors


  HTML
  
  <p>It's a paragraph.</p>  
  
  
  CSS
  
  p {
    color: yellow;
  }

It's a paragraph.

ID selectors


  HTML
  
  <p id="second">
    The second element.  
  </p>
  
  
  CSS
  
  #second {
    color: red;
  }

The second element.

Class selectors


  HTML
  
  <p class="latest">  
    Latest one.
  </p>
  
  
  CSS
  
  .latest {
    color: lime;
  }

Latest one.
