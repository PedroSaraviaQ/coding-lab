CSS selectors

Element selectors


  HTML
  
  <p>It's a paragraph.</p>  
  
  
  CSS
  
  p {
    color: yellow;
  }

It's a paragraph.

ID selectors


  HTML
  
  <p id="second">
    The second element.  
  </p>
  
  
  CSS
  
  #second {
    color: red;
  }

The second element.

Class selectors


  HTML
  
  <p class="latest">  
    Latest one.
  </p>
  
  
  CSS
  
  .latest {
    color: lime;
  }

Latest one.

Descendant selectors


  HTML
  
  <p>  
    This one is <em>good.</em>  
  </p>
  
  
  CSS
  
  p em {
    color: orange;
  }

This one is good.

Child selectors


  HTML
  
  <div>  
    <span>This one is a child.</span>  
    <p>
      This one is <span>not.</span>
    </p>  
  </div>
  
  
  CSS
  
  div > span {
    color: blue;
  }
This one is a child.

This one is not.

Pseudo-class selectors


  HTML
  
  <h4>Hover over here</h4>  
  
  
  CSS
  
  h4:hover {
    color: grey;
  }

Hover over here
